Plotting utilities¶

Plotting utilities.

equadratures.plot.plot_2D_contour_zonotope(mysubspace, minmax=[- 3.5, 3.5], grid_pts=180, show=True, ax=None)[source]¶

Generates a 2D contour plot of the polynomial ridge approximation.

Parameters

  • mysubspace (Subspace) – An instance of the Subspace class.

  • ax (matplotlib.ax) – An instance of the matplotlib axes class to plot onto. If None, a new figure and axes are created (default: None).

  • minmax (list) – A list of the minimum and maximum values of \(M^T x\), where \(M\) is the subspace.

  • grid_pts (int) – The number of grid points for generating the contour plot.

  • show (bool) – Option to view the plot.

equadratures.plot.plot_decision_surface(PolyTree, ij, ax=None, X=None, y=None, max_depth=None, label=True, color='data', colorbar=True, show=True, kwargs={})[source]¶

Plots the decision boundaries of the PolyTree over a 2D surface.

Parameters

  • PolyTree (PolyTree) – An instance of the PolyTree class.

  • ij (list) – A list containing the two dimensions to plot over. For example, ij=[6,7] with plot over the 6th and 7th dimensions in X.

  • ax (matplotlib.ax) – An instance of the matplotlib axes class to plot onto. If None, a new figure and axes are created (default: None).

  • X (:numpy.ndarray) – A numpy ndarray containing the input data to plot.

  • y (:numpy.ndarray) – A numpy ndarray containing the output data to plot.

  • max_depth (int) – The maximum tree depth to plot decision boundaries for.

  • label (bool) – If True then the subdomains are labelled by their node number.

  • color (string) – What to color the scatter points by. 'data' to color by the X,``y`` data. 'predict' to color by the PolyTree predictions, and 'error' to color by the predictive error. (default: 'data').

  • colorbar (bool) – Option to add a colorbar.

  • show (bool) – Option to view the plot.

  • kwargs (dict) – Dictionary of keyword arguments to pass to matplotlib.scatter().

equadratures.plot.plot_orthogonal_polynomials(Parameter, ax=None, order_limit=None, number_of_points=200, show=True)[source]¶

Plots the first few orthogonal polynomials.

Parameters

  • Parameter (Parameter) – An instance of the Parameter class.

  • ax (matplotlib.ax) – An instance of the matplotlib axes class to plot onto. If None, a new figure and axes are created (default: None).

  • order_limit (int) – The maximum number of orthogonal polynomials that need to be plotted.

  • number_of_points (int) – The number of points used for plotting.

  • show (bool) – Option to view the plot.

Example:

import numpy as np
from equadratures import *

myparam = eq.Parameter(distribution='uniform', lower = -1.0, upper = 1.0, order=8, endpoints='both')
myparam.plot_orthogonal_polynomials()

equadratures.plot.plot_polyfit_1D(Polynomial, ax=None, uncertainty=True, output_variances=None, number_of_points=200, show=True)[source]¶

Plots a 1D only polynomial fit to the data.

Parameters

  • Polynomial (Poly) – An instance of the Polynomial class.

  • ax (matplotlib.ax) – An instance of the matplotlib axes class to plot onto. If None, a new figure and axes are created (default: None).

  • uncertainty (bool) – Option to show confidence intervals (1 standard deviation).

  • output_variances (numpy.array) – User-defined uncertainty associated with each data point; can be either a float in which case all data points are assumed to have the same variance, or can be an array of length equivalent to the number of data points.

  • show (bool) – Option to view the plot.

equadratures.plot.plot_sufficient_summary(mysubspace, ax=None, X_test=None, y_test=None, show=True, poly=True, uncertainty=False, legend=False, scatter_kwargs={}, plot_kwargs={})[source]¶

Generates a sufficient summary plot for 1D or 2D polynomial ridge approximations.

Parameters

  • mysubspace (Subspace) – An instance of the Subspace class.

  • ax (matplotlib.ax) – An instance of the matplotlib axes class to plot onto. If None, a new figure and axes are created (default: None).

  • X_test (:numpy.ndarray) – A numpy ndarray containing input test data to plot (in addition to the training data). Must have same number of dimensions as the training data.

  • y_test (:numpy.ndarray) – A numpy ndarray containing output test data to plot (in addition to the training data).

  • show (bool) – Option to view the plot.

  • poly (bool) – Option to plot the subspace polynomial.

  • uncertainty (bool) – Option to show confidence intervals (1 standard deviation) of the subspace polynomial.

  • legend (bool) – Option to show legend.

  • scatter_kwargs (dict) – Dictionary of keyword arguments to pass to matplotlib.scatter().

  • plot_kwargs (dict) – Dictionary of keyword arguments to pass to matplotlib.plot().
