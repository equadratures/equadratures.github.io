.. meta::
   :description: equadratures is an open-source python code for uncertainty quantification, surrogate-based optimisation, numerical integration and data-driven dimension reduction. It is built on orthogonal polynomial approximations.
   :keywords: polynomials, polynomial chaos, optimisation, orthogonal polynomials, dimension reduction


``equadratures`` is the data science platform of choice for thousands of data scientists. Itâ€™s easy to use in desktop, cloud, or cloud-in-a-box environments. What separates ``equadratures`` from other platforms is its ability to work on both large and small datasets and offer readily interpretable models.

Given it's novel algorithmic core, it is purpose built to offer end-to-end intelligent and robust data solutions. It is also flexible enough to be integrated with standard data workflows.

.. rubric:: Usage

The code can be used in isolation to analyse an existing database, e.g., a ``.csv`` file of data.


.. figure:: /_documentation/Figures/schematic_2.png
   :width: 100%

Alternatively it can be used within a workflow where both inputs and outputs feed into other packages, e.g., a fluid or structural simulation package.

   :width: 100%
